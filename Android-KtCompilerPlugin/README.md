
# KCP


### KCP 相比 KSP 的缺点
1. API not stable.
2. Complex api.
3. Allow modifying existing code.
4. not easy
5. not fast


### 相比 KSP 的优点
1. Modify funcs & classes
3. APT is Jvm only , but kcp not



### Annotation Processor Vs KCP
APT
1. Works in compile time
2. emit source
3. work on java or kotlin

KCP
1. Works in compile time
2. emit bytecode
3. work ONLY on kt
